# qq_maj_bot
查询雀魂账号信息，希望可以实现：


1. 提供用户名/用户ID，(--pt/-p)查询用户的pt数据（包括四麻和三麻），实现类似thsbot的效果；
2. 提供用户名/用户ID，提供段位房间（--room/-r 王/王座/玉/金）（缺省为最后一场段位的查询），提供查询场次（--count/-c num）（缺省为50场，范围在1-5000），实现类似thsbot的效果（和铳）；
3. 提供用户名/用户ID，（-d/--detail），查询详细数据（包括平均打点/铳点，立直和率/铳率，放铳至立直等）（同样可以通过-c/-r进行筛选）；
4. 提供用户名/用户ID，（-b/--bind）绑定用户，类似thsbot；
5. 提供用户名/用户ID，查询用户最近场数mortal4.1b版本上的rating（--find/-f）
6. 提供用户名/用户ID，查询用户连打情况，筛选近特定场次中，1天内的最大连打数和平均连打数，并计算间隔时间(--time/-t)
7. (--game/-g begin) 雀兜，开始后获取用户输入例如-g 123m123p123s11177z
未开始输入将提示：输入-g begin或--game begin开始游戏；
非和牌型将给出提示：并非合法和牌型！
正确将提示：答对了！
不正确时生成图片：绿色/橙色/灰色代表相应信息
输入-g end结束并公布答案
8. --help/-h 提供以上内容信息


由于通过牌谱屋实现，故大概率仅支持雀杰及以上段位的数据查询。

又，很可能不会实现通过id查询的逻辑。
又又，估计短时间不会提供天凤账号查询。
